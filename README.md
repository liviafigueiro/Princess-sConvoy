# Princess-sConvoy
Projeto desenvolvido em Python utilizando a biblioteca Pygame, com foco na criação de um jogo 2D.

Durante o desenvolvimento, fortalecei minha lógica de programação, organização e estruturação de código, além de adquirir maior familiaridade prática com a linguagem Python. O projeto também contribuiu para o desenvolvimento de habilidades como resolução de problemas, interpretação de erros e aprendizado contínuo por meio de pesquisa e documentação.
